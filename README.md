# 📡 Telecommunication Customer Churn Prediction System

## 📌 Project Overview

This project is a Machine Learning based web application that predicts whether a telecommunication customer is likely to stay or churn based on customer information.

The application is developed using Python, Flask, HTML, CSS and a Decision Tree Classifier.

---

## 🚀 Features

- Predict customer churn
- Clean and simple user interface
- Decision Tree Machine Learning model
- Real-time prediction
- Model accuracy displayed
- Professional web application using Flask

---

## 🛠 Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Joblib
- HTML
- CSS

---

## 🤖 Machine Learning Algorithm
